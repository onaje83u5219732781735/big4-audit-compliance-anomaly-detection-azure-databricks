# 08 – Security & Compliance

See also `SECURITY.md` and `ETHICS.md`.

Key aspects:

- IAM and RBAC via Azure AD groups.
- VNet integration and private endpoints for all data services.
- Encryption using Key Vault–managed keys (CMEK).
- RLS/CLS for BI and SQL views exposing risk scores.
- Purview catalog and lineage for regulatory & audit requirements.
